# DT Drag & Drop

DT Drag & Drop is a React library for dealing with drag and Drop inventory modal.

## Installation

```bash
npm install
```

## Usage

```
React Js
React Js functional Components with Hooks.
Use Redux or any other middleware to notify about the component changes.
```
